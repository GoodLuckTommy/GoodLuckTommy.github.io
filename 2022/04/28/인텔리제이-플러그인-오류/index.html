인텔리제이 플러그인 검색 안되는 오류

2022-04-28
  • intellij

# 증상

인텔리제이 마켓 플레이 검색이 안되는 현상.

인터넷은 문제 없었고, 설정의 Http Proxy 설정해봐도 안됨.

# 원인

인텔리제이이 최적화를 위해 추가한 Custom VM Option값이 문제였다. 설정시 추가한 값중에 Djsse.enableSNIExtension=false가 원인

# 해결

Djsse.enableSNIExtension옵션에 true값을 주거나 옵션을 삭제

# Djsse.enableSNIExtension이란?

대략 SNI를 지원하냐 마냐 옵션. SNI(Server Name Indication)는 통신 프로토콜로 tcp통신중 핸드 세이크 과정에서 클라이언트가 어느 호스트명에 접속하는지 서버에 알리는 역할을 한다. 서버는 한개이나 도메인이 여러개일 경우 같은 서버인지 파악하는 용도. 즉, 하나의 웹서버가 여러 웹사이트를 서비스 하면 인증서 인증에 관한 문제가 발생되는데, 이를 해결하기 위해 기존 TLS 프로토콜을 확장했다고 한다. 결국 인텔리제이 마켓 서버와 통신 프로토콜 문제
