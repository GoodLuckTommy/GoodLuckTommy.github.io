인텔리제이 자동 빈줄 추가

2022-04-22
  • intellij

인텔리제이를 사용하던 중 내가 넣지 않은 new line이 생기는 현상이 발생했다. '뭐지? 난 엔터 한번만 눌렀는데..' 빈라인을 삭제하니, 소스 한줄이 삭제되어 버렸다. 알고 보니 실제 빈라인이 아니고 인텔리제이에서 가상으로 만든 라인이었다.

Editor 설정등을 변경해 봤으나 여전..

# 원인

인텔리제이에서 제공하는 Inlay Hints (opens new window) 기능으로 인해 hint 보여주는 공간이 추가 된 것. 그럼 내가 친 엔터인지 니가 친 엔터인지 알 수 없잖아!?

요놈이 범인!

# Inlay hints 설정

안쓰기에는 아까운 기능이니, 힌트를 보여주는 위치를 변경해 보자. 설정으로 가서

Preferences > Editor > Inlay Hints

힌트 위치를 오른쪽으로 설정


힌트 종류마다 위치를 따로 설정 할 수 도 있다. 사용하지 않고 싶을땐 왼쪽 체크박스의 체크를 해제하자.
